# tastamp-light Roadmap

> 이 파일은 프로젝트의 단일 진실 공급원(Single Source of Truth)입니다.
> Claude(및 기타 AI 모델)는 세션 시작 시 이 파일을 읽고, 유저 지시 없이는 완료 체크 외 임의 편집을 하지 않습니다.

---

## 🎯 기획 의도 및 핵심 비전 (Core Vision)
* **타겟 및 목적:** '검색하는 방법'조차 모르는 이용자들에게 직관적이고 쉬운 정보를 제공하며, 술과 음식에 진심인 위스키 애호가들의 맛집(보물창고)을 모으고 공유하기 위함입니다.
* **기존 매체의 한계 극복:** 유튜브, 블로그 등 기존 매체의 '정적이고 수동적인 정보 갱신'과 '위치 체감의 어려움'을 해결합니다.
* **해외여행 특화:** 해외 검색 및 동선 계획의 극악한 난이도를 보완하여, 누구나 지도를 통해 직관적으로 위치를 파악하고 동적인 정보를 얻을 수 있도록 설계합니다.

## 🚧 진행 중

### 🛡️ 운영 및 커뮤니티 강화 (Domestic)

**구현 방향:** 유저 참여형 데이터 검증 및 관리자 운영 효율화

**태스크:**
- [x] 디스코드 신고 웹훅: 장소/댓글 신고 시 관리자 디스코드 채널 알림
- [x] Vercel Web Analytics 연동 (트래픽 추적)
- [x] 기여자(Contributor) 표시: 장소 카드/상세에 `Added by [닉네임]` 명시
- [x] 소셜 리액션: 장소별 직접적인 '좋아요/싫어요' 기능
- [ ] 장소 한정 MD/보틀 연동 (Phase 2): 증류소 전용 핸드필(Hand-fill)이나 DE(Distillery Exclusive) 바틀 등 해당 장소에서만 구할 수 있는 특산품/한정판 표시 기능

---

### 🌏 해외 지역 지원 (Global Expansion)

**구현 방향:** 지도 렌더링 비용을 절감하는 '리스트 기반 MVP' 전략

**세부 논의:**
- 해외 탭 지도: **무료 Google Maps Embed API**로 표시(무제한 0원) — 선택 장소를 핀으로, 미선택 시 국가 개요. 인터랙티브 JS/Static Maps는 트래픽 보고 추후 판단 (2026-06-14 유저 결정: "명색이 지도 앱이니 지도는 떠야 한다, 우선 무료로")
- 국가별 1개씩만 조회·표시(모든 국가 동시 로딩 폐기) — 로딩 리소스 절감
- 구글 맵 앱 아웃링크(딥링크)는 상세의 '지도보기' 버튼으로 병행 유지
- 월별 해외 장소 저장 횟수 제한(Quota) 기반 구독제($2~3) 도입

**태스크:**
- [x] `src/lib/adapters/`: IPlaceStandard 인터페이스 및 API 파서 구축
- [ ] DB 스키마 업데이트: `region`, `source`, `usage_quota` 컬럼 추가 (region·source 기존 존재, usage_quota 미적용)
- [x] 구글 Places API 세션 토큰 및 필드 마스크 최적화 적용 (Autocomplete+Details, includedRegionCodes로 한국 밴)
